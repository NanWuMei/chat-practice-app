# 🪞 镜子 — 情感聊天练习系统

> 你以为你在跟 AI 聊天，其实你在照镜子。

一个本地运行的中文聊天训练工具。与 AI 角色模拟微信聊天，通过苏格拉底式的自我提问发现自己的聊天模式——不评分、不给答案，只帮你看见自己。

## 快速开始

```bash
git clone https://github.com/NanWuMei/chat-practice-app.git
cd chat-practice-app
npm install
cp .env.example .env   # 编辑 .env 填入你的 API Key
npm start              # 一键启动前端+后端
```

启动后打开 http://localhost:5173，选择角色即可开始。

> **API Key 获取：** 前往 [小米 MiMo 开放平台](https://open.xiaomimimo.com) 注册，创建 API Key 填入 `.env` 的 `MIMO_API_KEY`。

## 使用流程

```
选择角色 → 创建分身 → 新建会话 → 聊天 → 结束并复盘
                                              ↓
                                     苏格拉底提问（你回答）
                                              ↓
                                     写下行动锚点（可选）
                                              ↓
                                     查看历史模式（5次后解锁）
```

1. **选择角色** — 首页选择练习对象（目前内置梁友安）
2. **创建分身** — 每个分身有独立的关系状态，可以从头练习
3. **开始聊天** — 像微信一样自然地聊，AI 角色会根据关系温度调整回复风格
4. **结束并复盘** — 系统自动检测关键时刻（6种类型），苏格拉底针对每个时刻提问
5. **写下反思** — 回答苏格拉底的问题，写下你的发现
6. **行动锚点** — 写下"下次想试的一个小变化"（可选）
7. **模式发现** — 5次复盘后解锁，按场景分类回顾你的历史反思

## 设计理念

- **不评分** — 没有 1-10 分，没有"打败 80% 用户"
- **不给改法** — 不会说"你应该改成 XXX"
- **只提问** — 苏格拉底式的开放式问题，帮你自己发现
- **看见自己** — 你的模式、你的节奏、你的盲区

## 内置角色

### 练习对象

| 角色 | 来源 | 特点 |
|---|---|---|
| 梁友安 | 《爱情而已》 | 27岁体育经纪人，独立、有边界感、对套路免疫 |

基于电视剧角色深度蒸馏，包含 5 个心智模型、8 条决策启发式、完整表达 DNA。根据交互共振分数动态调整回复风格。

### 复盘提问者

| 角色 | 身份 | 作用 |
|---|---|---|
| 苏格拉底 | 哲学家 | 复盘时通过开放式提问帮你自我审视 |

基于柏拉图对话录等 2000 年研究传统蒸馏，6 个心智模型、8 条决策启发式。不评判、不给建议，只追问。

## 关键时刻类型

| 类型 | 含义 |
|------|------|
| KM_A | 她回复特别长（投入度上升） |
| KM_B | 她回复特别短（投入度下降） |
| KM_C | 你分享了自己的经历/感受 |
| KM_D | 你连续追问但没有分享 |
| KM_E | 她主动发起新话题 |
| KM_F | 对话突然中断 |
