# 🪞 镜子 — 情感聊天练习系统

> 你以为你在跟 AI 聊天，其实你在照镜子。

一个本地运行的中文聊天训练工具。与 AI 角色模拟微信聊天，通过苏格拉底式的自我提问发现自己的聊天模式——不评分、不给答案，只帮你看见自己。

## 快速开始

```bash
git clone https://github.com/NanWuMei/chat-practice-app.git
cd chat-practice-app
npm start
```

启动后浏览器自动打开 http://localhost:5173，选择角色即可开始。

> **首次运行需要配置 AI API：** 编辑项目根目录的 .env 文件，填入你的 API Key。
> 支持任意 OpenAI 兼容服务商（小米 MiMo、DeepSeek、OpenAI 等）。
> 详细步骤见 [SETUP.md](./SETUP.md)。

## 使用流程

```text
选择角色 → 创建分身 → 新建会话 → 聊天 → 结束并复盘
                                              ↓
                                     回溯上次锚点（有则追踪执行情况）
                                              ↓
                                     苏格拉底提问（你回答）
                                              ↓
                                     罗杰斯聚焦器（镜像摘要 + 行动选项）
                                              ↓
                                     确认行动锚点（选一个或自定义）
                                              ↓
                                     查看历史模式（5次后解锁）
```

1. **选择角色** — 首页选择练习对象（目前内置梁友安）
2. **创建分身** — 每个分身有独立的关系状态，可以从头练习
3. **开始聊天** — 像微信一样自然地聊，AI 角色会根据关系温度调整回复风格
4. **结束并复盘** — 系统自动检测关键时刻（6种类型），苏格拉底针对每个时刻提问
5. **写下反思** — 回答苏格拉底的问题，写下你的发现
6. **罗杰斯聚焦器** — AI 生成镜像摘要 + 2-3 个精准行动选项，降低填写锚点的认知负担
7. **确认锚点** — 选择一个选项或自定义写下"下次想试的一个小变化"
8. **模式发现** — 5次复盘后解锁，按场景分类回顾你的历史反思

## 设计理念

- **不评分** — 没有 1-10 分，没有"打败 80% 用户"
- **不给改法** — 不会说"你应该改成 XXX"
- **只提问** — 苏格拉底式的开放式问题，帮你自己发现
- **看见自己** — 你的模式、你的节奏、你的盲区

## 内置角色

### 练习对象

| 角色 | 来源 | 特点 |
|---|---|---|
| 梁友安 | 《爱情而已》 | 27岁体育经纪人，独立、有边界感、对套路免疫 |

基于电视剧角色深度蒸馏，包含 5 个心智模型、8 条决策启发式、完整表达 DNA。根据交互共振分数动态调整回复风格。

### 复盘引导者

| 角色 | 身份 | 作用 |
|---|---|---|
| 苏格拉底 | 哲学家 | 复盘时通过开放式提问帮你自我审视 |
| 卡尔·罗杰斯 | 人本主义心理学家 | 聚焦器：生成镜像摘要和精准行动选项 |

- **苏格拉底** — 基于柏拉图对话录等 2000 年研究传统蒸馏，6 个心智模型、8 条决策启发式。不评判、不给建议，只追问。
- **卡尔·罗杰斯** — 基于罗杰斯全部著作、临床录音转录蒸馏，6 个核心心智模型、8 条决策启发式。像一面精准的镜子，反映行为模式而非评价。

## 聚焦器

v3.2 新增的核心功能。在苏格拉底提问结束后、填写行动锚点之前，罗杰斯会生成：

- **镜像摘要** — 一句话描述你这次聊天的行为模式
- **2-3 个行动选项** — 每个选项包含具体场景触发条件 + 行为动作，用第一人称"我"表述
- **自定义输入** — 保留自由表达空间

输入源（双轨）：
- **主体：对话记录** — 消息频率、追问次数、自我暴露次数、关键时刻类型
- **修饰：上次锚点 + 本次反思** — 校准选项的难度和语气

## 关键时刻类型

| 类型 | 含义 |
|------|------|
| KM_A | 她回复特别长（投入度上升） |
| KM_B | 她回复特别短（投入度下降） |
| KM_C | 你分享了自己的经历/感受 |
| KM_D | 你连续追问但没有分享 |
| KM_E | 她主动发起新话题 |
| KM_F | 对话突然中断 |
